# Vehicle CRUD API

This is a .NET-based API that allows performing CRUD (Create, Read, Update, Delete) operations on a vehicle entity. The API is designed to be simple, scalable, and easy to integrate with other systems.

---

## Features

- **Vehicle CRUD**: Allows creating, reading, updating, and deleting vehicle records.
- **Database**: Uses SQL Server to store data.
- **Entity Framework Core**: Implements the Repository pattern with Entity Framework Core for data access.
- **Swagger**: Automatic API documentation using Swagger UI.

---

## Requirements

- [.NET SDK 9.0.200](https://dotnet.microsoft.com/download/dotnet/9.0)
- [SQL Server](https://www.microsoft.com/en-us/sql-server/sql-server-downloads) or a compatible database.
- [Visual Studio](https://visualstudio.microsoft.com/) or [Visual Studio Code](https://code.visualstudio.com/) (optional).
